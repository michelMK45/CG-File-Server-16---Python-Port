# CGFS 16 Server 16 Python Port v0.1.0

Initial public release of the Python port of CGFS 16 Server 16 for FIFA 16.

## Highlights

- Python-based rewrite of the original FIFA 16 Server 16 workflow
- Windows desktop control panel and in-game overlay support
- Stadium, scoreboard, TV logo, movie, chants, and camera management
- Assignment dialogs and settings editors
- Public project structure prepared for community contributions
- PyInstaller packaging for a standalone executable release

## Included Asset

- `Server16Python.exe`

## Notes

- This is intended as the first public community release.
